## VMware Services (Windows)

Aplicación para iniciar y detener los servicios de VMware Player con un solo clic. Solo para Windows.

### Descarga

| Tipo | Archivo | Descripción |
|---|---|---|
| Instalador | **[VMware-Services-Setup-v2.1.0.exe](https://github.com/enigmoes/VMware-Services/releases/download/v2.1.0/VMware-Services-Setup-v2.1.0.exe)** | Instala en Program Files con acceso directo |
| Portable | **[VMware-Services-portable-v2.1.0.zip](https://github.com/enigmoes/VMware-Services/releases/download/v2.1.0/VMware-Services-portable-v2.1.0.zip)** | Sin instalación, extrae y ejecuta |

> Requiere Windows 10 / 11 x64 y VMware Player instalado. No requiere tener Java instalado.
